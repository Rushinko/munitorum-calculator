## Munitorum Calculator
This tool is used to calculate combat probability for Warhammer 40k 10th edition.  

![Munitorum Calculator](/public/calculator.png)
Create datasheets you would like to test.
Add modifiers like "Sustained Hits 1", or "Devestation wounds".
Calculate the results.
![Results](/public/results.png)
See data for exact rolls, or passes.
Calculate multiple weapon profiles at once, or multiple datasheets to determine which is right for your list.
